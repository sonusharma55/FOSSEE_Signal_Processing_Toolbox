digitrevorder

Calling Sequence

[y,i] = digitrevorder(x,r)
y = digitrevorder(x,r)

Parameters

x:

Vector of real or complex values

r:

radix / base

y:

input vector in bit reverse order

i:

indices after bit reversing

Description

This function returns the input data after reversing the digits of the indices and reordering the elements of the input array.

Examples

x = [%i,1,3,6*%i] ;
r = 2 ;
[y i]=digitrevorder(x, r)
